# Appel à tests et retours / Call for testing and feedback

**🇫🇷 Français**

Ce projet est une expérimentation cryptographique. Il n’a pas été audité et ne doit pas être utilisé pour protéger des données réelles.

Néanmoins, nous invitons la communauté à :

- Tester l’implémentation sur différentes plateformes (Linux, macOS, Windows via WSL).
- Vérifier les résultats des tests unitaires (`make test`).
- Mesurer les performances sur vos propres machines (`make bench`).
- Analyser la construction mathématique (permutation sponge, intégrale de contour, utilisation de ζ(s), etc.).
- Suggérer des améliorations, signaler des bugs, proposer des optimisations.

**Comment contribuer ?**

- Clonez le dépôt : `git clone https://github.com/PLATON-y/pctamalou-complex-cipher.git`
- Compilez et testez (voir `README.md`).
- Ouvrez une issue sur GitHub pour rapporter un problème ou une idée.
- Soumettez une *pull request* avec vos corrections ou améliorations.

**Merci de votre aide !**

---

**🇬🇧 English**

This project is a cryptographic experiment. It has not been audited and should not be used to protect real data.

Nevertheless, we invite the community to:

- Test the implementation on different platforms (Linux, macOS, Windows via WSL).
- Verify the unit test results (`make test`).
- Benchmark performance on your own machines (`make bench`).
- Analyze the mathematical construction (sponge permutation, contour integral, use of ζ(s), etc.).
- Suggest improvements, report bugs, propose optimizations.

**How to contribute?**

- Clone the repository: `git clone https://github.com/PLATON-y/pctamalou-complex-cipher.git`
- Build and test (see `README.md`).
- Open an issue on GitHub to report a problem or an idea.
- Submit a pull request with your fixes or improvements.

**Thank you for your help!**
